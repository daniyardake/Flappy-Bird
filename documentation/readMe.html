Barrier

Barrier is the wall that can kill Flappy Bird. Barriers will be later created as an array list.

Barrier
+rect: Recrangle
+height: int
 

 

Pseudocode:

Barrier{
create green rectangle of size 20x20;
}

 

Bird

Bird is a square that is manipulated by speed point. Speed point is (kinda) vector, which showing direction in which bird should move. Also, depending on moveUp or moveDown image of rectangle changes.
 
 
Bird
+ speed: Point2D
+ rect: Rectangle
+ lost: boolean
+ imgUp: Image
+ imgDown: Image
+ imageUp: ImagePattern
+ imageDown: ImagePattern
+ moveY(y: int ): void
+ moveX (x: int ): void
+ jump(): void
 
Pseudocode:
 
Bird{
create point
create rectangle
create boolean value
create two images

Create Bird()

moveX: move x coordinate of rectangle
loop through barriers
if barrier intersect bird->lost
if bird passed barier -> beep() and score++
move x coordinate

moveY:
if input value>0 then moveDown is true and set image downFlag
else moveDown is false and set image upFlag

move y coordinate of rectangle
loop through barriers
if barrier intersect bird->lost

if y coordinate <0 set it 0
if y coordinate >580 (which is 600-20) then lost

move y coordinate
}
